## My process
I follow a simple yet effective workflow utilizing jira to maintain steady progress.
Design[:white_check_mark:] -> Develope -> Test -> Done
Luckily Front-end Mentor provides the designs :see_no_evil:

#### Work flow

- Review the design 
- Create project files 
- Initialize a git repo
- Configure gulpfile.js
- Init a package.json
- Structure HTML
- Base CSS
- Mobile first development
- css for breakpoints 
- Add functionality with js
- Test for cross browser compatibility
- README documentation 
- Deploy

### What I learned

I wanted to start on a simple layout to implement grid. I could have easily done this in flexbox which I love. Grid was a little difficult to grasp initially but I will continue to use it for my projects moving forward to increase my comfortability with it.
